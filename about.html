Languages

HTML5

Content & Design

HTML5 is a markup language used for structuring and presenting content on websites. HTML5 determines the content displayed and location of the content on the site.

CSS3

Design

CSS3 is a style sheet language used for describing the presentation of a document written in a markup language. It is used to determine the placement of elements and basic effects.

Bootstrap

Design

Bootstrap is an open-source front-end web framework for designing websites and web applications. It allows HTML and CSS elements to be used in a way that makes the site responsive on all devices.

AngularJS

Development

AngularJS is a complete JavaScript framework used to integrate custom front-end development elements into the site. It allows for the implementation of a secure client or company intranet.

jQuery

Development

jQuery is a JavaScript library that allows custom integration of JavaScript effects and elements in specific parts of HTML. jQuery is very compatible with all devices, thanks to being a lightweight language.

MongoDB

Development

MongoDB is an open-source, cross-platform, document-oriented database program. In more simple terms, it's arguably the most scalable way to store a database of users for a client or company intranet.

Rest API

Development

A REST API provides seamless interoperability between systems on the Internet. For instance, if you have a merchant account through PayPal or Stripe, it can be woven into your site with their API.

WordPress

Content, Design & Development

WordPress is an open-source content management system based on PHP and MySQL. If you're looking to revamp your existing WordPress site, or start a new site, contact our experts.

PHP

Development

PHP is a server-side language primarily used for web development, but also used as a general-purpose programming language. It's most commonly used in contact forms and WordPress plugins.

Swift

Content, Design & Development

Swift is the latest general-purpose programming language developed by Apple®. We use Swift when developing your app because of its compatibility with the latest iOS devices.


Frequently Asked Questions

1 Where are you based out of?

The Altitude Web Services team is based solely out of Denver, CO (hence the "Altitude" in our name). None of our team members are based in other countries, which explains our quality of work.

2 Do you provide ongoing maintenance or support?

We can provide ongoing monthly or quarterly maintenance and updates for any site work we have completed. It's just $30/hour for content and design maintenance, and $60/hour for any development maintenance.

3 How does the flat pricing structure work?

Since many of our clients request very similar work, we decided to offer these core services in menu format, with the total end-cost visible up-front. That way there are no surprises.

If you are ordering from our core services menu, you only pay when the project is complete to your satisfaction. If we are doing an hourly custom service, we send a detailed invoice the third week of each month.

4 Do you offer a money back guarantee?

If you are ordering from our core services menu, you only pay when the project is complete to your satisfaction. So you never have to worry about receiving an invoice if you aren't satisfied.

If we are doing a custom hourly service, we ensure that the finished project is to your satisfaction, regardless of how many revisions it requires. And we never go over an initial quote for a custom service.

5 Do you work with existing Dev teams?

Absolutely! It's not uncommon for us to work with an existing programmer - or even a team of programmers. If you have a small add-on project and don't want to bring someone on full-time, hiring us can be the easiest, most cost-effective way of getting that project done.
